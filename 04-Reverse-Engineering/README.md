# ⚙️ Reverse Engineering

Reverse engineering involves analyzing compiled binaries, bytecode, or hardware firmware to deduce their logic, reconstruct algorithms, and extract secret flags.

## 🛠️ Essential Tools

- **Ghidra**: NSA open-source SRE framework (Decompiler & Disassembler)
- **IDA Pro / Free**: Interactive disassembler
- **Radare2 / Cutter**: Command line and GUI reversing framework
- **GDB + GEF/Pwnbg**: Dynamic debugging and execution tracing
- **pyinstxtractor / uncompyle6 / decompyle++**: Python bytecode decompilation
- **JD-GUI / Jadx**: Java `.class` and Android APK decompilation
