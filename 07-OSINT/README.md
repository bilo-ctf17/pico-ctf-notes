# 🕵️ OSINT (Open Source Intelligence)

This category focuses on publicly available information gathering, image geolocation, domain / IP history analysis, and digital footprint tracking.

## 🛠️ Essential Tools

- **Google Dorking**: Advanced search operators
- **Sherlock**: Social media account enumeration
- **Reverse Image Search**: Google Lens, TinEye, Yandex
- **Wayback Machine**: Historical website snapshots
